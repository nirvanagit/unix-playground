## Creating coredumps

Creating coredumps using TRACEBACK is only supported Linux, and not supported on macOS yet.

Set ulimit to unlimited
```
ulimit -c unlimited
```

Create binary
```
➜  coredump git:(master) ✗ ls
main.go   readme.md
➜  coredump git:(master) ✗ cd coredump
➜  coredump git:(master) ✗ go build .
```
